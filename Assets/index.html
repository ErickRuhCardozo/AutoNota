Cadastrando Usuários

O programa AutoNota permite que você cadastre seu usuário do Nota Paraná para que o login seja feito automáticamente, incluindo encerrar sessões iniciadas e fazer logout automático.

Diálogo Usuários Cadastrados

Para cadastrar um novo usuário, abra o diálogo de usuários, e clique no botão "Adicionar Usuário". Isso abrirá o diálogo de cadastrar usuário:

Interface do AutoNota

Preencha todas as informações corretamente. O campo "Nome" pode receber qualquer nome, não sendo necessário colocar seu nome do Nota Praná.

Tanto o Nome como CPF devem ser únicos nos registros do AutoNota. O programa não irá permitir cadastrar um novo usuário com um Nome ou CPF já cadastrados.

Após ter preenchido todos os campos, clique em "Cadastrar". O novo usuário irá aparecer no diálogo Usuários Cadastrados, de onde você poderá logar com ele.

Logando um Usuário

Para logar com um usuário no Nota Paraná, basta selecionar o usuário no diálogo "Usuários Cadastrados" e clicar no botão "Logar Com Usuário Selecionado". Após isso, o dialogo vai fechar e retornar para o programa, onde iniciará a tentativa de login.

Sessão Iniciada

O programa irá detectar automáticamente sossões iniciadas e finaliza-las, basta aguardar. Assim que a sessão do Nota Paraná de um usuário for finalizada, o programa irá fazer o login novamente.

Doando Notas

Após ter um usuário logado com sucesso, o programa irá fazer automáticamente a doação das notas escaneadas, respeitando o tempo limite do Nota Paraná. É importante frisar que o AutoNota não burla nenhum limite estabelecido pelo sistema do Nota Paraná. Isso significa que você não pode "acelerar" a doação de notas (por exemplo, não ter de esperar os 4 segundos de processamento da nota).

O Objetivo principal do AutoNota é que você possa escanear rápidamente suas notas e deixar que o programa doe-as para o Nota Paraná, sem que você tenha que ficar aguardando uma nota ser completamente processada para então escanear outra.

Dessa maneira, você pode escanear todas as suas notas e deixar o programa funcionando em segundo plano, enquanto usa o PC ou faz alguma outra atividade, sem ter de se incomodar.

Para escanear as chaves de acesso, basta deixar o campo "Chave de Acesso" com foco, isto é, selecionado, e escanear o código QR com uma máquina leitora ou através de uma webcam com auxílio de um software decodificador de QR Code.

Essa parte é muito importante pois o programa não recebe informações diretamente de máquinas leitoras ou programas de leitura de QRCode. Portanto, esse campo de texto deve sempre estar com foco (selecionado), de outra maneira, as notas escaneadas não serão armazenadas pelo programa e lançadas no Nota Paraná.

Configurações

No diálogo de configurações você pode informar o CNPJ da entidade para qual vai doar as notas e também, (opcionalmente) escolher um usuário para ser logado automáticamente quando você abrir o programa (Usuário Padrão).

Diálogo de Configurações

Caso tenha selecionado um usuário padrão e deseje reverter essa opção, basta clicar no botão ao lado da caixa de opções. Isso fará com que nenhum usuário logue automáticamente toda vez que o programa for aberto.
